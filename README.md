# html-to-pdf

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Demo
아래와 같이 convert 버튼을 클릭하면, 사전에 설정한 chunk size 단위로 pdf 가 분할 생성 및 다운로드 된다.
진행률도 확인할 수 있으며, pdf 가 chunk size 단위로 생성 완료될 때마다 진행률이 업데이트 된다.
